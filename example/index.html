Sobre

Consiste em uma adaptação e um conjunto de correções para a biblioteca Select2 (https://select2.github.io/).

Atualmente temos duas funções que basicamente funcionam utilizando jQuery:
1. $('select#selectAjax').select2Ajax(); - onde devemos informar alguns atributos data na   declaração do objeto HTML:
    data-url="" - url que será pesquisada
    data-placeholder="" - texto que fica por default no input
    data-limiteCaracteres="" - quando select multiplo, define o tamanho dos itens selecionados na sua exibição (quantidade de caracteres)
  OBS: Utilizando o AJAX, sua url requisitada receberá uma requisição POST com o input de nome "pesquisa" com o texto digitado pelo usuário para que seja adaptado os resultados.

2. $('select#selectAjax').select2Ajax(placeholder, semPesquisa); - onde pode se omitir os parâmetros placeholder e pesquisa, no entanto placeholder define o texto default do campo e semPesquisa quando true desabilita o mecanismo de pesquisa do campo.
